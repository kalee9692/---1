
# Lotto Number Generator

## Overview

This project is a simple web application that generates random lottery numbers. It's built with HTML, CSS, and JavaScript, and it's designed to be a fun and easy-to-use tool for anyone who wants to try their luck at the lottery.

## Features

*   **Lottery Number Generation:** The application generates 6 random numbers between 1 and 45, plus an additional event number.
*   **Visually Appealing Design:** The application has a modern and visually appealing design, with a dark theme, animations, and a clean layout.
*   **Responsive Design:** The application is responsive and works well on both desktop and mobile devices.
